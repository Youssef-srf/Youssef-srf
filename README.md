### 🏆 Featured Projects

#### 🧬 [CFTR Variant Classifier (Bio-informatics & AI)](https://github.com/Youssef-srf/CFTR-variant-classifier)
> **Stack:** Python, Scikit-Learn (Random Forest), Bio-informatics Algorithms
- A complete pipeline analyzing CFTR gene variants related to CBAVD. Combines sequence alignment (Needleman-Wunsch / Smith-Waterman) with Machine Learning to classify variants as Pathogenic, Benign, or Uncertain.

#### ⚕️ [Suivi-Derma (Healthcare PWA)](https://github.com/Youssef-srf/Suivi-derma)
> **Stack:** JavaScript ES6+, HTML5, CSS, Firebase, Service Workers
- A Progressive Web App designed for dermatology tracking. Features an offline-first architecture, visual photographic journals with automatic compression, and an interactive medical timeline for patient monitoring.

#### 🔐 [BlockRH (Decentralized HR & Blockchain)](https://github.com/Youssef-srf/BlockRH)
> **Stack:** Solidity, Ethereum, JavaScript
- A Decentralized Application (DApp) for HR management focusing on data integrity. Implements strict Role-Based Access Control (RBAC) via Smart Contracts to ensure privacy and security of sensitive records.

#### 📈 [Academic Success Predictor](https://github.com/Youssef-srf/Recommandation-Academique-IA)
> **Stack:** Python, Streamlit, Scikit-Learn
- An interactive predictive system built to forecast student success rates, featuring a clean Streamlit interface for real-time inference.
